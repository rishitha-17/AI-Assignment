## STEP 1: Install Dependencies
Open PowerShell in this folder and run:

    pip install -r requirements.txt

This installs: pydantic-ai, logfire, google-generativeai, pydantic

-------------------------------------------------------------------------------

## STEP 2: Authenticate with Logfire
Run this command:

    logfire auth

Follow the browser prompts to complete authentication.

-------------------------------------------------------------------------------

## STEP 3: Run the Agent

### OPTION A: Interactive Mode (Recommended for Your Assignment)

    python interactive_agent.py

Then you can type questions like:
- "Calculate 25 * 4 + sqrt(144)"
- "Search for artificial intelligence"
- "What is photosynthesis in science?"
- "Analyze sentiment: This is amazing!"
- "What is the current time?"

Type 'quit' when done.

### OPTION B: Single Query

    python single_query.py "What is 100 + 200?"

Or just:

    python single_query.py

Then enter your query when prompted.

### OPTION C: Demo (See All Features)

    python demo_research_agent.py

This runs 10 test queries showing all tools.
