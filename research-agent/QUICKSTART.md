# RESEARCH AGENT - QUICK START GUIDE

## Prerequisites
1. Python 3.10+ installed
2. Gemini API key configured (already done in code)
3. Logfire authentication

## Setup Steps

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Authenticate with Logfire
```bash
logfire auth
```
Follow the prompts to complete authentication.

### 3. Run the Agent

#### Interactive Mode (Chat with Agent)
```bash
python interactive_agent.py
```

Example queries to try:
- "Calculate 25 * 4 + sqrt(144)"
- "Search for artificial intelligence in technology"
- "What is photosynthesis?"
- "Analyze sentiment: This is wonderful!"
- "What is the current date?"

Type 'quit' to exit.

#### Single Query Mode
```bash
python single_query.py "Calculate 100 + 200"
```

#### Demo Mode (See all features)
```bash
python demo_research_agent.py
```

## Available Tools

1. **calculate** - Math operations
   Example: "Calculate pow(2, 10)"

2. **search_database** - Search knowledge base
   Categories: science, technology, history, general
   Example: "Search for Python programming"

3. **get_current_info** - Get system info
   Example: "What is the current time?"

4. **analyze_data** - Text analysis
   Example: "Count words in: hello world"

## Troubleshooting

### Error: "No module named 'pydantic_ai'"
Solution: Run `pip install -r requirements.txt`

### Error: "Logfire not authenticated"
Solution: Run `logfire auth`

### Error: "Gemini API error"
Solution: Check internet connection
